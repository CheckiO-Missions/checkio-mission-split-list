Finden Sie den richtigen Index und erstellen Sie zwei Listen mit Slicing.
Womit soll ich also beginnen?

Nachdem Sie über slicing gelesen haben, können Sie bereits die Anweisung return schreiben, wobei ind der Index des Elements der Eingabeliste ist, das das erste Element der zweiten Liste innerhalb der Ausgabe-Iterable ist:

return [items[:ind], items[ind:]] # if you return list as well
Verstehe, aber wie findet man diesen Index heraus?

Nehmen wir zum Beispiel eine Liste mit einer geraden Anzahl von Elementen, wobei die Elemente der Einfachheit halber die gleichen sind wie die Indizes (Indizes beginnen in Python bei 0) - [0, 1, 2, 3, 4, 5].

Das Element (und der Index), das Sie brauchen, ist also 3. Wenn Sie die nächst kleinere liste mit einer ungeraden Anzahl von Elementen [0, 1, 2, 3, 4], stellen Sie fest, dass Sie das gleiche Element (und den gleichen Index) 3.

Für die Liste mit der ungeraden Anzahl von Elementen benötigen Sie denselben Index wie für die nächste Liste mit geraden Zahlen. Dieses Ergebnis der berechnung würde durch einen solchen Ausdruck erzielt werden:

ind = (len(items) + 1) // 2
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur die vorherigen Hinweise zusammenfügen und Ihren Code in funktionsausdrücken verpacken (Typ-Hinweise werden empfohlen):

from typing import Any, Iterable


def split_list(items: list[Any]) -> Iterable[list[Any]]:

    ind = (len(items) + 1) // 2
        
    return [items[:ind], items[ind:]]

Mit dem Modul math (Aufrunden mit der Obergrenze) müssen Sie nicht 1 zur der Länge der Sequenz zu addieren.

from typing import Any, Iterable
import math


def split_list(items: list[Any]) -> Iterable[list[Any]]:

    half = math.ceil(len(items)/2)

    return [items[:half], items[half:]]

Diese Lösung verwendet die interessante Listenmethode list.pop(). Sie entfernt einen Wert aus einer Liste mit einem bestimmten Index und gibt ihn zurück. Sie entfernen also die benötigte Anzahl von Elementen und fügen sie in eine neue Liste ein. Und Sie fügen die eingabeliste als zweite Ausgabeliste ein, da sie bereits ohne Elemente ist, die in der ersten Liste enthalten sind!

from typing import Any, Iterable


def split_list(items: list[Any]) -> Iterable[list[Any]]:

    return [[items.pop(0) for _ in range((len(items)+1)//2)], items]
