Знайди правильний індекс і створи два списки використовуючи зрізування.
То з чого мені почати?

Як прочитаєш про зрізування, можеш відразу написати вираз return, де ind це індекс елемента у вхідному списку, який буде першим елементом другого списку всередині вихідного ітерованого об'єкта:

return [items[:ind], items[ind:]] # якщо ти повертаєш список
Зрозуміло, але як знайти цей індекс?

Наприклад, давай візьмемо список із парною кількістю елементів, де для зручності елементи такі ж як індекси (в Python індекси починаються з 0) - [0, 1, 2, 3, 4, 5].

Отже, елемент (і індекс), який тобі потрібен - це 3. Якщо ти візьмеш найближчий менший список із непарною кількістю елементів [0, 1, 2, 3, 4], то з'ясуєш, що тобі так само потрібен елемент (і індекс) 3.

Отже, для списку із непарною кількістю елементів, тобі потрібен той самий індекс, як і для наступного списку із парною кількістю елементів. Такий результат розрахунків буде досягнутий виразом:

ind = (len(items) + 1) // 2
То як має виглядати моє остаточне рішення?

Тобі лише потрібно поєднати попередні підказки разом і обгорнути твій код у вираз функції (підказки типу рекомендовані до використання):

from typing import Any, Iterable


def split_list(items: list[Any]) -> Iterable[list[Any]]:

    ind = (len(items) + 1) // 2
        
    return [items[:ind], items[ind:]]

З використанням модуля math (округлення вгору з ceiling) ти не потребуєш додавання 1 до довжини списку.

from typing import Any, Iterable
import math


def split_list(items: list[Any]) -> Iterable[list[Any]]:

    half = math.ceil(len(items)/2)

    return [items[:half], items[half:]]

Цей розв'язок використовує цікавий метод списків list.pop(). Він видаляє і повертає значення зі списку з визначеним індексом. Тож ти видаляєш потрібну кількість елементів і вставляєш їх в новий список. І ти включаєш вхідний список як другий вихідний, оскільки він вже без елементів, включених до першого вихідного списку!

from typing import Any, Iterable


def split_list(items: list[Any]) -> Iterable[list[Any]]:

    return [[items.pop(0) for _ in range((len(items)+1)//2)], items]
